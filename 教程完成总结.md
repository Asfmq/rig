## 💡 核心要点

### Agent 编排的 5 种模式（重点内容）

1. **Agent-as-Tool** 🔧
   ```
   子Agent → 包装成工具 → 主Agent调用
   ```

2. **Orchestrator** 🎯
   ```
   主Agent → 协调 → 多个专业Agent
   ```

3. **Router** 🔀
   ```
   输入 → 分类 → 路由到对应Agent
   ```

4. **Parallel** ⚡
   ```
   任务 → 同时发送给多个Agent → 合并结果
   ```

5. **Sequential** 🔄
   ```
   Agent1 → Agent2 → Agent3 → ... → 最终结果
   ```
